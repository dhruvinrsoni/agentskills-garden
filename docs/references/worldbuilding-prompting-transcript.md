## Core Thesis

> "A well-crafted prompt acts as a set of physical laws for a small universe."

Effective prompting = worldbuilding for AI. The same principles that make writing persuasive to humans make prompts effective for LLMs: start with shared ground (atomic units), build progressively, use in-group language, root with examples, maintain consistency, and respect cognitive limits.
